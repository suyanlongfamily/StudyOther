自己学Git的命令总结
0、查看命令，寻求帮助。很实用！
git --help

1、想查询有几个远程库
git remote -v

2、添加远程库
git remote add <远程库别名> <url> 

3、删除已经存在的远程库
git remote rm <远程库别名>

4、日志查询
git log

5、查看所有分支
git branch -a

6、创建分支
git branch <分支名>

7、推送到远程分支
git push <远程库别名>  <分支名>

8、删除本地分支
git branch -d <分支名>

9、删除远程分支
git branch -r -d <远程库别名/分支名>

10、切换分支
git checkout <分支名>

11、查看库的状态，是否有修改提交文件，以及要添加的文件。
git status -s
"??" 表示未知版本号文件。即可能是需要添加的文件。

12、添加文件到git暂存区。
git add <文件名>

13、提交做的修改并备注信息。然后可以通过 git log查询日志
git commit -m 

14、更新操作，即更新远程库的最新修改。
git pull命令远程命令来同步自己的本地仓库。
